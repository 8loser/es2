# Run

```
docker run --rm -ti tintin /bin/sh
```

如果有用 tmux

```
tmux -u
```

進入 tintint++

```
tt++ player.tin
```

載入角色

```
#read beggar.tin
```

# 檔案

- es2.map 地圖檔
- mapper.tin 畫地圖用
